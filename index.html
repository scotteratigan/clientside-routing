Client-Side Routing Step 4

Now that we have navigation working, let's navigate with links instead of a button.

For this to work, we will use regular anchor tags, but we will add a click listener to add custom behavior which will call our history.pushState function.

We will use a custom data property called data-customlink to identify our special links. But we will attach a click listener to the document so that any newly loaded links will function as expected as well.
